# Security Audit Checklist (Starter)

## Purpose
A simple checklist I can use to review basic security controls for a small business or home setup.

## 1) Access & Identity
- [ ] Strong passwords or passphrases used
- [ ] MFA enabled on key accounts (email, admin, banking)
- [ ] Unused accounts removed/disabled
- [ ] Admin access limited (least privilege)

## 2) Device & Patch Management
- [ ] OS updates enabled (Windows/macOS)
- [ ] Browser and common apps updated
- [ ] Antivirus/Defender enabled and updated
- [ ] Disk encryption enabled where possible (BitLocker/FileVault)

## 3) Network & Wi-Fi
- [ ] Router admin password changed from default
- [ ] Wi-Fi uses WPA2/WPA3 (not WEP)
- [ ] Guest Wi-Fi enabled for visitors
- [ ] Firmware checked/updated

## 4) Email & Phishing
- [ ] Staff know how to identify phishing signs
- [ ] Clear process to report suspicious emails
- [ ] Do not open unknown links/attachments

## 5) Backups & Recovery
- [ ] Backups scheduled (at least weekly)
- [ ] One backup copy kept separate (offline/cloud)
- [ ] Restore test performed (basic check)

## 6) Logging & Monitoring (Basic)
- [ ] Login alerts enabled where available
- [ ] Unusual activity reviewed when alerts happen
- [ ] Incident contacts documented
